# 💬 Real-Time Chat Application

A real-time web-based chat application built using **Flask**, **Socket.IO**, **HTML**, **CSS**, and **JavaScript**.  
This project enables multiple users to join a chat room and communicate instantly without page reload.

---

## 🚀 Features

- 🔄 Real-time messaging using WebSockets (Socket.IO)
- 👥 Multiple users support
- 🟢 User join/leave notifications
- 💬 Live chat updates without refreshing
- 🎨 Clean and responsive UI design
- ⚡ Fast and lightweight Flask backend

---

## 🛠️ Tech Stack

**Frontend:**
- HTML5
- CSS3
- JavaScript

**Backend:**
- Python
- Flask
- Flask-SocketIO

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/chat-app.git

2. Go to project folder
cd chat-app
3. Create virtual environment (optional but recommended)
python -m venv venv
4. Activate environment

Windows:

Mac/Linux:
